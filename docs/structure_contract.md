# Structure Contract (文档结构规范)

## Document Types

### Type A: Formal Book (正式书籍)
Based on 《一人公司：把自己当成一家公司来经营》 (100 pages)

### Type B: Action Guide (行动指南)
Based on 《一人公司0~100W行动指南》 (6 pages)

---

## Type A: Formal Book Structure

### Front Matter (前置部分)

1. **Cover Page (封面)**
   - Blue background, gold title, illustration, author, publisher info
   - No page number

2. **Table of Contents (目录)**
   - Page starts with "目 录" centered heading
   - Dot-leader format with page numbers right-aligned
   - Chapter entries bold, section entries indented
   - Format: "第一章 [page]" then "  第一节 [page]"

3. **Copyright Page (版权信息)**
   - "版权信息" centered heading
   - "COPYRIGHT" in small caps below
   - Fields: 书名, 作者, 出版社, 出版时间, ISBN, 字数
   - Publisher authorization statement
   - "版权所有 · 侵权必究"

4. **Endorsements (本书赞誉)**
   - "本书赞誉" centered heading
   - 6-10 testimonial quotes from industry figures
   - Each quote: paragraph of praise, followed by "——姓名 头衔"
   - Right-aligned attribution with em-dash

5. **Preface (序言)**
   - "序言" centered heading
   - Author name as sub-heading (e.g., "阿猫")
   - Personal narrative: author's journey and motivation
   - 2-4 pages of flowing text
   - Author name and date at bottom-right

### Body Matter (正文部分)

#### Chapter Structure (6-Chapter Model)

**第一章 商业认知** (Business Awareness)
- 第一节: 什么是一人公司
- 第二节: 如何打造一家一人公司
- 第三节: 实现一人公司需要具备哪些动机和方法
- 第四节: 把自己当成公司来经营
- 第五节: 再小的个体，也可以是一家公司

**第二章 流量能力** (Traffic Acquisition)
- 第一节: 流量：一人公司的起点
- 第二节: 获取私域流量的方法
- 第三节: 私域流量如何变现
- 第四节: 私域流量容易踩的三个坑

**第三章 产品能力** (Productization)
- 第一节: 为什么要做自己的产品
- 第二节: 如何把能力产品化
- 第三节: 如何提升产品价值
- 第四节: 如何用产品思维做内容
- 第五节: 把能力变成产品的四种形式
- 第六节: 如何从零启动第一个付费社群
- 第七节: 如何构建自己的产品体系

**第四章 营销能力** (Marketing & Sales)
- 第一节: 心理关卡：推广害怕被人骂怎么办
- 第二节: 价值输出：打造一个干货满满的朋友圈
- 第三节: 促单营销：推动用户下单的最后一步
- 第四节: 活动营销：用事件撬动销售
- 第五节: 裂变营销：如何利用群发售进行裂变推广

**第五章 生态能力** (Ecosystem Building)
- 第一节: 构建生态，十倍赋能
- 第二节: 如何找到自己的生态位
- 第三节: 如何用互推打造人际关系生态
- 第四节: 用合作思维搭建百万社群生态
- 第五节: 一人公司的终极目标是打造自己的生态圈

**第六章 IP能力** (Personal IP)
- 第一节: 个人IP的机会与价值
- 第二节: 尽早地完成IP的商业化
- 第三节: 如何用内容打造人设
- 第四节: 如何迈出创业第一步
- 第五节: 从身份（ID）到个人品牌（IP）的转变
- 第六节: 如何持续打造盈利的IP

#### Chapter Opener Page Format
Each chapter begins with a dedicated opener page:
```
        第X章        (centered, medium size)

    [Chapter Title]   (centered, large, bold)

  ■ Key takeaway 1
  ■ Key takeaway 2
  ■ Key takeaway 3
  ■ Key takeaway 4
```
- Black square bullet points (■) for chapter highlights
- 4-5 key takeaways summarizing the chapter
- Rest of page is whitespace

#### Section Format
```
        第X节        (centered)

    [Section Title]   (centered, bold)

[Body text paragraphs...]
```

#### Subsection Format
Subsection headings are left-aligned, bold, inline with text flow:
```
一人公司的概念

一人公司指的是，以一人为主体打造商业闭环...
```

### Back Matter (后置部分)

1. **Afterword (后记)**
   - "后记" centered heading
   - Co-author name as sub-heading (e.g., "鱼堂主")
   - Reflective personal narrative
   - 1-2 pages

2. **Reader Messages (附录)**
   - "觉醒品牌用户送给阿猫和鱼堂主的话"
   - Collection of reader testimonials

## Type B: Action Guide Structure

### Cover Section
- Large photo at top
- Title: 一人公司0~100W行动指南
- Author intro with credentials
- "使用必读:" section

### Module Structure (8 Modules)

**一、基础认知篇**
- 00 职场真相
- 01 赚钱思维
- 02 一人公司
- 03 真实案例

**二、方向选择篇**
- 01 个人梳理
- 02 市场分析（含案例）
- 03 明确起步策略

**三、流量篇**
- 01 公域流量
- 02 私域流量

**四、获客篇**

**五、产品设计篇**

**六、交付篇**

**七、商业篇**

**八、心力篇**

### Guide Style Elements
- Each major module has emoji callout box with key insight
- Yellow highlighted text for emphasis
- Checkbox (☑) items for article listings
- Red bold text for critical emphasis
- Module divider with bold Chinese numeral (一、二、三...)

---

## Figure Numbering

- Book style: "图X-Y" format (e.g., 图1-1, 图2-3)
- Caption centered below figure: "图1-1 一人公司的核心理念"
- Figures are referenced in text as "（如图1-1所示）"

## Page Numbering Rules

| Section | Page Number | Position |
|---------|-------------|----------|
| Cover | None | -- |
| TOC | Roman or none | bottom-right |
| Front matter | Roman (i, ii, iii) or none | bottom-right |
| Body | Arabic (1, 2, 3) | bottom-right |
| Chapter openers | None | -- |
