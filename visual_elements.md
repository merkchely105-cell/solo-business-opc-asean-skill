# Visual Elements (图表展示样式)

## Diagram Types & Specifications

### 1. Triangle Concept Model (三角概念模型)

**Purpose**: Show three core interconnected concepts
**Example**: 低成本 / 小规模 / 商业闭环

**Visual Style**:
- Three circles arranged in an equilateral triangle
- Light blue fill (#E3F2FD) with blue border (#64B5F6)
- Chinese text label centered in each circle
- Thin blue lines connecting the three circles
- Caption below: "图1-1 一人公司的核心理念"

**Layout**:
```
        [ 低成本 ]
           ◯
          / \
         /   \
        /     \
[ 小规模 ]◯———◯[ 商业闭环]
```

### 2. Circular Five-Force Model (五力循环模型)

**Purpose**: Show five interconnected business capabilities
**Example**: 流量能力 → 产品能力 → 营销能力 → 生态能力 → IP能力

**Visual Style**:
- Five circles arranged in a pentagon/circular pattern
- Light blue fill with blue border
- Chinese text labels in each circle
- Curved arrows connecting circles in sequence
- Caption: "图1-4 一人公司的五力模型"

### 3. Process Flowchart (流程图)

**Purpose**: Show linear business process
**Example**: 流量→获客→产品→交付

**Visual Style**:
- Rectangular boxes with rounded corners
- Light yellow or light blue fill
- Bold Chinese text inside boxes
- Arrow connectors between boxes
- Can be horizontal or vertical layout

### 4. Mind Map / Tree Structure (思维导图)

**Purpose**: Categorize knowledge or show hierarchical relationships
**Example**: Chapter content overview

**Visual Style**:
- Central topic node at center or top
- Branch lines to sub-topics
- Clean, minimal styling
- Chinese labels at each node

### 5. Highlight Callout Box (高亮提示框)

**Book Style**:
- Light yellow/cream background (#FFF8E1)
- Slight border or no border
- Used for key formulas, principles, or important reminders
- Full width or indented

**Guide Style**:
- Yellow background with rounded corners
- Emoji icon at left (e.g., 🏖️ beach umbrella)
- Bold text, sometimes with red accent
- Key phrases highlighted in yellow background

### 6. Chapter Key Points (章节要点)

**Format**:
- Black square bullets (■ Unicode 25A0)
- 4-5 concise statements
- Positioned on chapter opener page
- Below the chapter title

Example:
```
■ 自媒体收入=产品价格×（私域数量×购买率）×推出产品的数量
■ 重点不是接广告的形式，重点是一定要接广告。
■ 销售自己的产品才是流量变现的最终阶段。
■ 信任是交易的基石。
```

### 7. Checkbox Item Lists (勾选列表)

**Format**:
- Unicode checkbox characters (☑ or ☐)
- Used for article lists in guide style
- Blue hyperlink-style text for titles

Example:
```
☑ 30岁前必须知道的职场真相
☑ 为什么说，上班不再是一个人发展的最佳路径
☑ 裸辞1年我才明白，做自己，才是一个人真正的事业
```

### 8. Quote Block / Testimonial (引用块)

**Format**:
- Indented text
- Em-dash attribution right-aligned
- Used for endorsements and expert quotes

Example:
```
这本书从流量、营销、产品、生态、个人IP多个环节切入，
帮助一些小团队打破误区...

                        ——亦仁 生财有术创始人
```

## Figure Caption Format

- All figures numbered as "图X-Y"
- Caption centered below figure
- Format: "图[章号]-[序号] [中文描述]"
- Referenced in text as: "（如图1-1所示）"

## Color Usage by Element

| Element | Primary Color | Background |
|---------|--------------|------------|
| Concept diagrams | Blue border #64B5F6 | Light blue fill #E3F2FD |
| Flowcharts | Blue border | Light yellow #FFF8E1 or light blue |
| Callout boxes | Yellow accent #FFC107 | Cream #FFF8E1 |
| Highlighted text | Red #E53935 or black | Yellow #FFEB3B |
| Chapter bullets | Black #000000 | None |
| Mind map nodes | Blue border | White or light blue |

## Illustration Style (封面插画)

- Cartoon/illustration style (not photography)
- Character juggling multiple tasks simultaneously
- Multiple arms representing multitasking
- Color palette matches cover: yellow/gold character on blue background
- Simple line art style, friendly and approachable
- Represents the "one person doing everything" concept

## Emoji Usage (Guide Style)

| Emoji | Usage |
|-------|-------|
| 🏖️ | Module insight/key takeaway callout |
| 💡 | Tips, ideas, important notes |
| ☑️ | Article/section listing |
| ⬜ | Unchecked item |
| 📌 | Pinned/important content |

## Diagram Creation Guidelines

1. **All labels must be in Chinese**
2. **Keep diagrams clean and uncluttered**
3. **Use consistent color scheme throughout**
4. **Ensure text is readable at print size**
5. **Center diagrams on page with whitespace around**
6. **Always include figure caption with proper numbering**
